How to compile :
1. Download all source
2. Download toolchain
3. Edit bbuil.sh to match with toolchain patch
4. Open terminal and go to kernel directory
5. chmod 777 bbuil.sh
6. Run bbuild.sh by use command ./bbuild.sh
7. Read and do it.
